# 创新脚手架 · 从想法到一页纸的三步工具包

不知道怎么创新？跟着走就行。三步，从零到一页纸。

## 你是谁

你可能是任何岗位的员工——产品、技术、运营、HR、市场。你不需要是创新专家。
这个工具包会一步步引导你。你只管回答问题，工具帮你整理思路。

## 核心定位

**这个工具包只做三件事：帮你搞懂战场、定义问题、想出方案。**

最终交付物是一张**立项一页纸**——拿着它去立项答辩、找老板聊、拉人组队。

做prototype（原型/产品）和test（用户验证/市场测试）不是这个工具包的事。那需要根据具体情况开发数字员工、做产品、找用户验证——是外部的事。

## 三步流程

基于 **Stanford d.school 设计思维**（Bootleg, 2018）：

```
Step 1  Empathize（共情）  行业研究 → 摸清用户和赛道
  ↓
Step 2  Define（定义）     定义问题 → 形成POV（Point of View）
  ↓
  ┌──── 方向验证gate ────┐
  │  POV值不值得深入？     │
  │  不值得 → 打回Step 2  │
  │  值得 → 继续          │
  └──────────────────────┘
  ↓
Step 3  Ideate（孵化）    发散方案 → 收敛到立项一页纸
```

每一步的产出是下一步的输入。你不需要从第一步开始——如果你已经有想法了，直接从Step 2开始。

---

## 每一步做什么

### Step 1：Empathize / 行业研究（共情）

**什么时候用**：你想创新但不知道哪个方向值得做。
**触发词**：「帮我调研一下XX」「行业扫描」「竞品分析」
**你会得到**：一份千字级行业扫描——发展脉络、竞品图谱、机会判断
**底层工具**：横纵分析法 + Playwright网页爬虫

### Step 2：Define / 定义问题（定义）

**什么时候用**：你有一些信息/想法，需要把问题说清楚。
**触发词**：「帮我把问题说清楚」「定义问题」「找到核心痛点」
**你会得到**：一份POV（Point of View）——你在解决谁的什么问题，为什么值得解决
**底层工具**：d.school POV框架 + How Might We问题

### 方向验证gate

**触发时机**：Step 2的POV形成后，进入Step 3之前。
**验证什么**：用四巨头轻量版（每人一句话快判）验证方向——
- 巴菲特：这个问题的护城河潜力在哪？
- 马斯克：这个问题本身该存在吗？还是在优化不该存在的东西？
- 纳瓦尔：解决这个问题需要谁的许可？杠杆够不够？
- Karpathy：这个问题有没有数据飞轮的潜力？
**不通过**：打回Step 2重新定义问题。
**通过**：进入Step 3发散方案。

### Step 3：Ideate / 发散方案（孵化）

**什么时候用**：问题定义清楚了（POV + 方向验证通过），需要想方案。
**触发词**：「帮我想方案」「头脑风暴」「写立项一页纸」
**你会得到**：一份结构完整的立项一页纸——问题、方案、商业逻辑、资源需求、第一步
**底层工具**：段永平商业模式过滤器 + 本分检查 + DT/千丁双模板

## 怎么用

### 方式一：跟着流程走（推荐新手）

1. 安装本toolkit到Claude Code
2. 说「我想做一个创新项目，从零开始」
3. 系统会引导你一步步走完三个阶段

### 方式二：按需使用

直接说你想做什么：
- 「帮我调研一下XX赛道」→ Step 1
- 「帮我把问题说清楚」→ Step 2
- 「帮我想方案 / 写立项一页纸」→ Step 3

### 方式三：从任意点进入

如果你已经有了调研结果，直接说「帮我把问题说清楚」，系统会从Step 2开始。

## 可选附加模块

以下模块不在核心三步流程中，但在需要时可以单独调用：

- **04-pitch**：路演材料生成器——把一页纸一键转化为HTML展示页面
- **05-review**：完整AI预审——四巨头完整版评审，适用于答辩前的深度验证

## 安装说明

### 环境要求
- Claude Code（已安装并可用）
- Node.js >= 18（Step 1行业研究的深度爬虫需要）
- Chromium浏览器（首次运行时自动安装）

### 首次安装

1. 将 `innovation-toolkit/` 文件夹复制到Claude Code的skills目录
2. 运行 `install.bat`（Windows）或 `install.sh`（Mac/Linux）
3. 在Claude Code中说「创新工具包」即可触发
